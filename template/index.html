UI Component Library

A modular component library. Each component is in its own file in the components/ folder.

Available Components:

  • ✅ Modal - Central responsive container
  • ✅ Tabs - Desktop + mobile navigation
  • ✅ Status Bar - Bottom fixed bar
  • ✅ Service Cards - Minimal service display
  • ✅ Service Modal - Detail popup
  • ✅ Portfolio Cards - Project showcase
  • ✅ Command Palette - Quick nav (Cmd+K)
  • ✅ Buttons - Various styles

How to Use:

  1. Open components/[component-name].html
  2. Copy the HTML structure
  3. Include css/theme.css in your page
  4. Include the JS file if the component needs it
