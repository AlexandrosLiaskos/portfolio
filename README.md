# Personal Portfolio & CV

A modern, responsive portfolio website and CV built with Rust (Actix-web) and styled with Tailwind CSS. Features a sleek dark theme with cyan accents and a clean, professional design.

## 🌟 Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Dark Theme**: Modern dark theme with cyan accents
- **Interactive UI**: Smooth animations and transitions
- **Print-Friendly CV**: Optimized CV layout for printing
- **Fast & Lightweight**: Built with Rust for optimal performance
- **SEO Friendly**: Proper meta tags and semantic HTML

## 🛠️ Tech Stack

- **Backend**: 
  - Rust
  - Actix-web
  - Tera Templates
  
- **Frontend**:
  - HTML5
  - Tailwind CSS
  - JavaScript
  - Font Awesome Icons
